## 基础语法
### 前置概念
1. Go属于C系函数,故也通过main函数启动:
```go
package main
import ("fmt")

func main() {
  fmt.Println("Hello World!")
}
```
2. Go中的所有文件都属于某个包中,需要使用package 关键字声明,并写在程序的第一行
